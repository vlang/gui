# Requirements: v-gui Gradient Rendering

**Defined:** 2026-02-01
**Core Value:** Gradients render correctly with arbitrary color stops — no visual artifacts

## v1 Requirements

### Linear Gradients

- [x] **LIN-01**: User can define linear gradient with 3+ color stops at arbitrary positions
- [x] **LIN-02**: User can specify gradient direction via CSS-compatible angles (45deg, 90deg, etc.)
- [x] **LIN-03**: User can specify gradient direction via CSS keywords (to top, to bottom-right, etc.)

### Radial Gradients

- [x] **RAD-01**: User can define circular radial gradient from center
- [x] **RAD-02**: User can define radial gradient with 3+ color stops at arbitrary positions

### Infrastructure

- [x] **INF-01**: Fragment shader replaces current vertex interpolation hack
- [x] **INF-02**: Dithering prevents visible color banding artifacts
- [x] **INF-03**: Metal shader (macOS) renders gradients correctly
- [x] **INF-04**: GLSL shader (Linux/Windows) renders gradients correctly
- [x] **INF-05**: Color interpolation uses premultiplied alpha (CSS spec compliance)
